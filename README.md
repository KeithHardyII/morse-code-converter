# Morse Code Converter

A simple desktop application built with **Python** and **Tkinter** that converts text to Morse code and Morse code back to text.

---

## Features

- Convert plain English text to Morse code
- Convert Morse code back into readable text
- Handles letters, numbers, and common symbols
- Simple and clean GUI built with Tkinter

---

## Technologies Used

- Python 3
- Tkinter (standard Python GUI library)

---

## How to Run the Project

1. Clone or download the repository
2. Make sure Python 3 is installed
3. Run the program with:

```bash
python main.py
```

---

## Example Usage

**Text → Morse:**

```
HELLO
```

Becomes:

```
.... | . | .-.. | .-.. | ---
```

**Morse → Text:**

```
.... . .-.. .-.. ---
```

Becomes:

```
HELLO
```
